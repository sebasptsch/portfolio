# This is a Test Blog Post

This is a sample blog post to demonstrate the structure and formatting of a markdown file in a static site generator.

## Introduction

Welcome to this test blog post. Here, we will explore various features of markdown and how they can be used to create engaging content.

## Features

- **Headings**: Use `#` for headings. More `#` symbols indicate lower-level headings.
- **Lists**: Create ordered and unordered lists using `-`, `*`, or numbers
- **Links**: Add links using `[link text](URL)`
- **Images**: Embed images using `![alt text](image URL)`
- **Code Blocks**: Use triple backticks for code blocks:
```python
def hello_world():
    print("Hello, world!")
```
- **Blockquotes**: Use `>` for blockquotes
